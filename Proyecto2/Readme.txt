En cada carpeta se encuentra un fichero con la solucion de cada parte

Para compilar utilizar gcc file.c -pthread

Para ejecutar utilizar ./a.out N M

Con N un entero que representa la cantidad de hilos
Y con M un entero que representa la cantidad de etapas. 

El programa imprimira secuencialmente cada etapa, con el id del hilo que se ejecuto y un numero random asociado al hilo.
